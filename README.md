# Pong Football — учебная 2D-игра в стиле пинг-понг с футбольными воротами на Python (Pygame)

![Демонстрация!](images/screen_shot.jpg)


# Описание:
Это небольшая учебная игра, сочетающая механику классического пинг-понга с футбольной тематикой. Игрок управляет вратарем, двигаясь строго по вертикали, защищая большие ворота, расположенные за левой границей (для левого игрока) и за правой границей (для правого игрока). Цель — отбивать мяч и не допускать гола.

# Фишка проекта: 
Каждый раз, когда игрок успешно отбивает летящий мяч, на поле появляется временный напарник (teammate). Он стоит на месте и, при контакте с мячом, отбивает его так же, как основной игрок, после чего исчезает. Это создает интересную динамику.

# Управление:
Esc — выход из игры

W/S — вверх/вниз (Player 1)

↑/↓ — вверх/вниз (Player 2)

# Установка 
Клонируй репозиторий:


```bash
git clone https://github.com/ваш_ник/Pong_football.git
cd Pong_football
```


Установи зависимости (если потребуется):
```bash
pip install pygame
```

Запусти игру:
```bash
python main.py
```
